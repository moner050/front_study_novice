자바스크립트의 데이터 타입과 기본 데이터 타입

자바스크립트의 변수는 모든 데이터 타입을 참조할 수 있다.(동적타입dynamically typed) 이 때 모든 프로그래밍 언어와 같이 자주 쓰이는 타입을 기본데이터 타입으로 지정하고 있으며 총 7가지가 있다.(string,number,bigint,boolean,symbol,null,undefined)

typeof Number 숫자형

정수와 (float)부동소수점, 무한대(Infinity), NaN(Not a Number) 숫자가 아님의 오류 나타낸다. 숫자형은 *,+,-,/ 등의 연산을 할 수 있다.

  • " 10 "/2 : (자동으로 형 변환한다." 10 "->10)

  • 객체지향 언어에서 객체는 new 연산자로 생성한다.(자바스크립트 new를 생략할 때가 있다.) 예)Number 클래스로 객체를 생성-> new Number()

    둘다 값이 정해지지 않은 상태로 undefined는 초기화 되지않은(선언되지않은) 상태고 null '타입'이 정해지지 않은 빈 상태

    기본데이터 타입은 아니지만 기본데이터 타입처럼 사용할 수 있는 배열과 json

    let arr=[a,b,c,d]; : new 없이 생성 가능한 배열타입 ( arr=new Array(a,b,c,d))

    배열은 순서를 가지는 데이터타입으로 모든 데이터 타입을 참조할 수 있다.( 같은 데이터 타입을 참조하는 것을 권장)

    let json={key:value,key2:value2} : new 없이 생성가능한 JSON 타입(new Object())

    json은 key와 value를 가지는 데이터 타입으로 new Object로 생성 가능하다. 이때 Object는 모든 클래스의 부모인 Object와 별개의 클래스다.
